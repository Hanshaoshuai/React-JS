react  通过ref 加 useRef   获取dom元素，如果该dom元素更改了  怎么去获取更改后的dom内容；可以将ref=useCallback函数添加依赖项获取新的dom：如下

const drawers = useCallback(
    (node) => {
      if (node !== null) {
       // 处理
      }
    },
    [list, filters],
  );

<div ref={drawers}></div>

1、子类必须在 constructor 方法中调用 super 方法，如果子类没有定义 constructor 方法，constructor 方法以及其内部的 super 方法会被默认添加。
2、在子类的 constructor 方法中，只有调用 super 之后，才可以使用 this 关键字，否则会报错。
3、在一个方法前加上关键字 static，就表示该方法不会被实例继承，但是父类的静态方法，会被子类继承。
	class A {
    		static say() {
        			console.log("hello");
    		}
	}
	class B extends A {}
	console.log(B.say()); //hello
